NAME

xe-tunnel-param-list - Displays all parameters for a tunnel

SYNOPSIS

xe tunnel-param-list uuid=<TUNNEL UUID>

DESCRIPTION

xe tunnel-param-list displays all parameters for a specific tunnel.

REQUIRED PARAMETERS

uuid

Tunnel UUID - Use xe tunnel-list to obtain a list of tunnel UUIDs.

EXAMPLES

To display all parameters for a specific tunnel:

  • xe tunnel-param-list uuid=<TUNNEL UUID>

SEE ALSO

xe help tunnel-param-list, xe-tunnel-create(1), xe-tunnel-list(1), xe-tunnel-param-set(1), xe-tunnel-param-remove(1) xe-pif-list(1), xe-network-list(1)
